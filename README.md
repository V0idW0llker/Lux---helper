# Lux — голосовой помощник

Локальный голосовой помощник для ПК на Python.

## Что умеет сейчас
- Здоровается голосом при запуске (утром, днём и вечером по-разному)
- Понимает текстовые команды из консоли и отвечает голосом:
  - привет, как дела, кто ты
  - время / который час
  - помощь / что умеешь
  - выход / пока

## Планы
- Распознавание речи с микрофона
- Команды: открыть браузер, заметки
- В будущем — подключение ИИ

## Запуск
```
pip install -r requirements.txt
python main.py
```
