# Lux — голосовой помощник

Локальный голосовой помощник для ПК на Python.

## Что умеет сейчас
- Своё окно (tkinter): история диалога, поле ввода и кнопка «Отправить»
- Слушает команды с микрофона (кнопка 🎤) и распознаёт русскую речь
- Здоровается голосом при запуске (утром, днём и вечером по-разному)
- Понимает текстовые команды и отвечает голосом:
  - привет, как дела, кто ты, спасибо
  - время / который час, дата / какой сегодня день
  - открой браузер / ютуб / блокнот / калькулятор / проводник
  - громкость: громче / тише / выключи звук
  - подбрось монетку, кинь кубик, погода
  - выключи / перезагрузи компьютер (с подтверждением)
  - помощь / что умеешь
  - выход / пока
- Команды вынесены в отдельный файл `commands.py` (таблица «фразы → действие»)
